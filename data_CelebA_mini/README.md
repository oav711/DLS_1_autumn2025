## mini CelebA

Подготовленный уменьшенный датасет CelebA, вырвненные изображения, распределение изображений по папкам для обучения моделей нейросетей, а также файлы метаданных размещены по ссылке:

**https://disk.yandex.ru/d/RDZxES6s1OgxkA**

## Структура датасета:

```
data_CelebA_mini
├── data\                 # отобранные изображения из полного датасета CelebA
│     └── *.jpg
├── data_aligned/         # обрезанные и выровненные изображения
│     └── *.jpg
├── data_fr/              # датасет для тренировки моделей распознавания
│     ├── train/          # тренировочная часть
│     │     └── *.jpg
│     ├── val/            # валидационная часть
│     │     └── *.jpg
│     ├── test/           # тестовая часть
│     │     └── *.jpg
├── data_IRM/             # датасет для оценки Identification Rate Metric 
│     ├── query/
│     │     └── *.jpg
│     ├── distractors/
│     │     └── *.jpg
├── metadata_fr/          # метаданные частей обучающего датасета
│     └── *.csv
└── *.csv                 # метаданные уменьшенного датасета (без разбиения на train/val/test/query/distractors)
```


_Примечание. Ввиду сложности загрузки множества мелких файлов папки data, data_aligned, data_fr заменены архивами *.7z. В случае желания проверки/оценки скриптов проекта их необходимо распаковать._
